# Basic HTML Project
This is a basic HTML project of a fictional tech company built with pure HTML.
This project contains multiple pages linked with each other.

## 1.Home Page of The project
! [App Screenshot] (images/home-page.png)

Home page contains 3 columns describing the three key features of the company.

## 2.Related Pages
### 2.1.About Page
About Page contains the mission and values of the company. Along with that contact details also provided.

### 2.2 Contact Page
In the contact page there is a fully accessible contact form and locaction of the company also given.

### 2.3. Service Page
Service Page contains the key services provided by the company and includes a comprehensive comparison table.

### 2.4 Team Page
This page provides an overview of the company's core team members.

## 3. Technologies Used
- **HTML** : HTML is used to give structure to the webpages.
